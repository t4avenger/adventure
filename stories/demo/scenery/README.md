# Scenery images for the demo story

Place image files here that your nodes reference via `scenery` in `demo.yaml`. The server serves only files in this directory.

- Use filenames with or without extension in YAML, e.g. `scenery: "forest"` or `scenery: "forest.png"`.
- Supported formats: `.png`, `.jpg`, `.jpeg` (server tries these if no extension is given).
- Example: for `scenery: "forest"` add `forest.png` (or `forest.jpg`) in this folder.
